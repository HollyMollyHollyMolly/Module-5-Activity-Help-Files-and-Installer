# Intermediate Software Development Activity 5

This activity will help to reinforce learning of the Module 5 concepts of:

- Generating Help Files
- Generating a Distributable Installer

## Author

Nguyen Dang Thai Ha

## Additional Information

This project is designed to reinforce concepts from Module 5 by providing hands-on experience with Python documentation, packaging, and application installation:

1. **Help Files** – Generating HTML documentation for Python classes using **Sphinx**.
2. **Installer** – Packaging the Python application into an executable and creating an installer using **PyInstaller** 
3. And installing **Inno Setup**.
